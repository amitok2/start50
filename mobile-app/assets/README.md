# Assets Directory

This directory contains static assets for the Base44 mobile app.

## Required Assets

### Images
- `icon.png` - App icon (1024x1024)
- `splash.png` - Splash screen image
- `adaptive-icon.png` - Android adaptive icon (1024x1024)
- `favicon.png` - Web favicon (48x48)

### Fonts
Place Hebrew fonts in `assets/fonts/`:
- `Heebo-Regular.ttf`
- `Heebo-Medium.ttf`
- `Heebo-Bold.ttf`

You can download Heebo fonts from Google Fonts:
https://fonts.google.com/specimen/Heebo

## Image Specifications

### App Icon (icon.png)
- Size: 1024x1024 pixels
- Format: PNG with transparency
- Content: Base44 logo on transparent or colored background

### Splash Screen (splash.png)
- Recommended size: 1284x2778 pixels (iPhone 14 Pro Max)
- Format: PNG
- Background: White or brand color
- Content: Base44 logo centered

### Adaptive Icon (adaptive-icon.png)
- Size: 1024x1024 pixels
- Format: PNG with transparency
- Safe zone: Keep important content within 66% diameter circle
- Content: Base44 logo

### Favicon (favicon.png)
- Size: 48x48 pixels (or larger, will be scaled down)
- Format: PNG
- Content: Simplified Base44 logo

## Notes

- All images should be optimized for mobile (compressed)
- Use consistent branding colors (rose, pink, purple)
- Maintain the web app's visual identity
- Test icons on both light and dark backgrounds
